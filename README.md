# dotfiles

### youtube-dl mp3 music download script
`ln -s pathtothisrepo/dotfiles/yt.py where/to/place/symlink`\
Run script as `python yt.py --url=<YOUTUBE_VIDEO_OR_PLAYLIST_URL>`\
See options `python yt.py --help`
